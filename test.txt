1. Заполнение формы "Будьте в курсе событий" некоректными данными:
   - Футер главной страницы
   - ввести почту без @ и заполнить все остальные поля
   - нажать кнопку "Подписаться"
 Получаем предупреждение: Пожалуйста, введите правильный адрес электронной почты (Пример: johndoe@domain.com).
 
 
2. Использовать фильтр для поиска товаров:
   - с главной страницы перейти в раздел "Женское"
   - выбрать любое значение в фильтре по типу 
   - выбрать любое значение в фильтре по цвету 
   - выбрать любое значение в фильтре по размеру
 Результат: список товаров отсортирован по данным критериям.
 
 
3. Добавление товара в корзину
   - с главной страницы перейти в раздел "Мужское"
   - нажать на фото выбранного товара
   - выбрать размер
   - нажать на кнопку "Положить в корзину"
   - нажать на иконку корзины
 Результат: в корзине появился выбранный товар
 

4. Ввод несуществующего промокода, для получения скидки
   - с главной страницы перейти в раздел "Мужское"
   - нажать на фото выбранного товара
   - выбрать размер
   - нажать на кнопку "Положить в корзину"
   - нажать на кнопку "Оформление заказа"
   - введем промокод 444
   - нажать кнопку "Применение промокода"
 Получаем предупреждение: Неверный код купона
 
 
5. Ввод несуществующего адреса доставки
   - с главной страницы перейти в раздел "Женское"
   - выбрать товар, его размер и добавить в корзину
   - нажать кнопку "оформить заказ"
   - ввести контактные данные
   - выбрать спопоб доставки курьером
   - выбрать спопоб оплаты при получении
   - выбрать условия и время доставки
   - введем адрес доставки: улица ывоалыт, дом 45234234, квартира 234
   - нажать кнопку "Оформить заказ"
 Результат: заказ был принят с некорректным адресом доставки
 
 
6. Поиск конкретного товара:
   - главная страница
   - в строке поиска вводим "Chuck"
   - нажимаем на кнопку поиска
 Результат: получили список товаров, которые содержат данное слово
 
 
7. Ввод пароля, состоящего из 3 символов, при регистрации:
   - на главной странице нажать Войти/Регистрация
   - нажать кнопку "Создать учетную запись"
   - запоолнить поля личной информации
   - заполнить поля для авторизации
   - в поле пароль ввести: 123
   - повторить пароль
 Получаем предупреждение: Минимальная длина этого поля должна быть равна или больше 8 символов. Пробелы перед и после символов будут проигнорированы.


8. Вход на сайт незарегистрированным пользователем:
   - на главной странице нажать Войти/Регистрация
   - введем в поле email "ogurtsova-01@mail.ru"
   - введем пароль от почты
   - нажать кнопку "Войти"
 Получаем предупреждение: Вы ввели некорректные данные или ваша учётная запись временно заблокирована.
 
 
9. Сортировать товары:
   - на главной странице нажать "Мужское"
   - в поле сортировка выбрать сортировку по цене
 Результат: список товаров был отсортирован по увеличении цены
 
 
10. Удаление единицы товара из корзины при наличии других заказанных товаров
    - на главной странице перейти в раздел "Женсокое"
	- нажать на иконку выбранного товара
	- выбрать размер
	- увеличить количество до 3
	- нажать кнопку "Добавить в корзину"
	- в шапке сайта нажать на "Мужское"
	- нажать на иконку выбранного товара
	- выбрать размер
	- нажать кнопку "Добавить в корзину"
	- нажать кнопку "Оформить заказ"
	- уменьшить количество товара с 3 до 2
 Результат: количество товаров уменьшилось
